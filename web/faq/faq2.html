FAQ: What is SneezyMUD?

This page last modified: 12-13-97

SneezyMUD is a MUD similar in some ways to other muds on the Internet. Our original source was SillyMUD DIKU code, but with the release of version 4.1, all original code will be in place.

SneezyMUD is written in C++ and utilizes object-oriented coding techniques. It has roughly 20 megabytes of code, 5 megabytes of database related information and 15 megabytes devoted to the player and rent files for the 1000+ characters. In total, more than 10000 manhours have been devoted to creating SneezyMUD by either the half dozen implementors or by one of the 50 or 60 builders we have had since to first SneezyMUD went public in June of 1992. There are 116000 lines of code at present.

All rights to the SneezyMUD code reside with the SneezyMUD Implementation Team. At present, no version of SneezyMUD code is available to the public and all contributions become the property of the SneezyMUD Implementation Team.

SneezyMUD remains committed to bringing the Internet the best possible mud without charge. It runs on a machine offered to us by the great folks at Stanford University. SneezyMUD's gods volunteer their time to improving the mud.
