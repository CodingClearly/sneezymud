The Following Books are Available as ascii text capture files:
**NOTE: The files are script generated from the book chapters files online. Unfortunately in order to make sure to include any new chapters they use a basic script with a standard wildcard. The way it works though causes the chapters to be out of order. What this means is the chaptes are in this order: 1*, 2*, 3*, ect. So a book with 31 chapters would be: 1, 10, 11, 12, 13, 14, ect, 2, 21, 22, 23, 24, ect, 30, 31.
  • begin.txt - Beginner's Guide to Building
  • time.txt - The Meaning of Time
  • redit.txt - Redit & World Creation
  • oedit.txt - Object File Creation
  • armor.txt - Armor Guidance
  • medit.txt - Medit & Mob Creation
  • zonefile.txt - Zonefile Creation
  • quest.txt - Constructing Response Quests
