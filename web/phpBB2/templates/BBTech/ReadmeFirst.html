BBTech Style template
- Flash fx (lite flash on left side of the header)
- Non flash version included (simply by editing 1 line.)
- Fits most resolution (600 x 800 , 1024 x 768)
- Cross browser compatibility (tested under IE 6.0, Mozilla firebird, Netscape, Opera 7)
Flash on or off.
By default, the flash is displayed, however, if you wish to use gif instead of flash. I have commented the line located under overall_header.tpl. Simply remove the flash params and uncomment the line above it to use gif instead.
Extras / Resources
Under the resources folder in this zip you will find The psd.(photoshop format) files is for you to add your own logo if you're using the non-flash version. The blank buttons psd are post buttons and profile buttons for you to edit if using another lang. Fla(maromedia flash mx2004 format) file is for you to edit the flash to the way you like. These files are extra and edit them at your own will.
Enjoy!
This is my first attempt at making style template for full version phpbb. Usually I do styles for the ported version for phpnuke. Hope you all enjoy it, feel free to let me know if any questions or suggestions you may have.
